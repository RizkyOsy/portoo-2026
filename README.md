## 📌 Deskripsi Proyek

Website portfolio personal ini dibuat sebagai project UTS mata kuliah **Pemrograman Web**. Website ini dirancang untuk memperkenalkan diri sebagai **Laravel Web Developer** sekaligus menampilkan daftar project yang pernah atau sedang dibuat.

Aplikasi ini dibangun menggunakan **Laravel**, **Livewire**, **Blade**, **Filament Admin Panel v3**, **MariaDB**, dan **Docker**. Website ini juga memiliki admin panel untuk mengelola data profile, project, progress project, serta pesan masuk dari contact form.
